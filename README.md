

 
  How to run:
    <li> first use the makefile (make).
    <li> ./server
    <li> ./client localhost
    <li> Commands:
    'PUSH' to push a string.
    'POP' to POP a string.
    'TOP' to show the last string.
    'STOP' to exit.
